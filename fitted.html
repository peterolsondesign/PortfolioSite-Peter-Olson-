Fitted

  • Organization: CareerFoundry
  • Duration: 4 months
  • Role: UX/UI Designer

Objective

Design an exercise web application which helps users start and maintain consistent exercise routines.

User Needs

  • Database of exercises and recommendations
  • Calendar to schedule workouts
  • Earn badges by completing preset or self-created goals
  • Way to connect with friends and share progress

Moodboard

Users want to feel energized and inspired while using Fitted. An orange-based color-scheme and active images in the moodboard convey this aspect.

Style Guide

Final Screens

Sign-up and Login

Here, users can select their fitness level, access their profile, and navigation via a hamburger menu.

Find and Add Exercises

Using a filter, users can access a collection of exercises. Each exercise includes a brief description, difficulty level, and a how-to video. User can also add exercises to their calendar.

Share Achievements

Fitted comes with a bunch of achievements to help motivate users improve their health. After completing an achievement, users recieve a badge which they can share with their friends!

Set Goals

Users can also add their own exercise, diet, and weight goals!
